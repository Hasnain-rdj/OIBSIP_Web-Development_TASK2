## ✨ Features

* **Interactive UI/UX:** A sleek design with glassmorphism effects, 3D hover animations, and smooth scrolling.
* **Light/Dark Mode:** A fully functional theme toggle that saves user preferences using `localStorage`.
* **Dynamic Animations:** Features a custom animated loading screen and a continuous typewriter effect for professional titles.
* **Responsive Design:** Completely optimized for desktops, tablets, and mobile devices using CSS Flexbox and CSS Grid.
* **Project Showcase:** Interactive project cards with quick links to live demos and GitHub repositories.
